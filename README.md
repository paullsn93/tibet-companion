# 西藏旅伴 Tibet Companion (v13.0) 🏔️

這是一款專為 12 日西藏深度旅遊設計的數位助手。結合了行程管理、實時氣候監測、高海拔健康追蹤以及專為攝影愛好者設計的民俗與攝影工具箱。

## 🌟 核心功能

### 1. 📖 互動式使用手冊 & 雲端行程 (v13.0)
- **前端入口整合**：首頁具備「開啟旅程」與「使用手冊」雙入口。
- **雲端行程連結**：移除本地詳細描述，改為透過 Smart URL Fragments 智慧連結至阿玩旅遊官網，確保行程資訊最即時。
- **高可讀性設計**：針對戶外環境優化的 **Extra Large** 字體 (內文 2xl) 與高對比度色票。

### 2. 📸 攝影與民俗工具 (Photo & Culture) - v11.0 升級
- **日出日落**：對接 Open-Meteo API，根據當前行程顯示精準的黃金時刻時間。
- **民俗禁忌**：整理 5 大必看禁忌（轉經方向、攝影尊重、摸頭禁忌等），提升旅途文化深度。

### 3. 🌤️ 實時氣候與海拔動態
- **動態氣象**：抓取實時氣溫與降雨機率。
- **海拔警告**：智慧辨識高海拔地區 (3500m+)，自動提示氧氣儲備與活動建議。

### 4. 🏥 健康監控中心 (Health Tracker)
- **血氧與心率**：每日記錄。
- **數據導出**：支援一鍵複製整日健康數據，方便發送給親友或醫護人員。

### 5. 🛠️ 實用工具箱 (Toolkit)
- **藏語對話**：具備藏式語調的文字轉語音 (TTS) 功能。
- **4-4-4 呼吸法**：動態圖形引導，協助緩解高原反應。
- **互動行李清單**：清單式管理，自帶本地保存功能。

## 🚀 快速開始

1. **本地執行**：
   使用 PowerShell 執行 `start_server.ps1` 啟動 8000 埠的本地伺服器。
2. **重置行程**：
   從封面點擊「開啟旅程 Enter」將自動重置至第一天行程。

## 🛠️ 技術棧
- **Frontend**: Vue 3 (Composition API)
- **Styling**: Tailwind CSS, Material Icons
- **API**: Open-Meteo (Weather & Astro)
- **Voice**: Web Speech API
